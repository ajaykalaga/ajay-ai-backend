I implemented JWT-based authentication, role-based access, and pagination. The API follows REST principles and is used by a frontend client.

Project: Blog Platform Backend
I developed a backend for a blogging platform using Django. Users can register, create posts, edit them, and comment. I implemented user authentication, permissions, and database models. The project helped me understand ORM, relationships, and query optimization.

Project: AI Portfolio Assistant (Ask Ajay)
I am building an AI-powered portfolio assistant using LangChain, RAG, and Streamlit. The system ingests my resume and project notes, stores them in a vector database, and allows users to ask questions about me. The bot answers only using my real data and avoids hallucinations. This project demonstrates my understanding of LLM APIs, embeddings, vector databases, and retrieval-augmented generation.
